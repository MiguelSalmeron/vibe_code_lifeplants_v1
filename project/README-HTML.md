# README HTML / Legacy Static Reference

## ES

Este archivo documenta la capa estática heredada del proyecto: los HTML de la raíz, los estilos en `styles/` y los scripts en `scripts/`. Hoy se conserva como referencia y compatibilidad histórica; la aplicación activa se documenta en [README.md](README.md).

### Qué incluye esta capa

- `index.html`
- `explorar-plantas.html`
- `styles/`
- `scripts/`
- `server.py` y `server.sh`

### Cómo ejecutarla

```bash
python3 server.py
```

Alternativas:

```bash
python3 -m http.server 8000
npx http-server -p 8000
```

### Estado de la capa HTML

- Es útil para revisar el legado visual y de contenido.
- No es el flujo principal de desarrollo del repositorio.
- No debe confundirse con la app React Router activa.
