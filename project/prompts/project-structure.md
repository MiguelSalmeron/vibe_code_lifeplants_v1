# Project Structure
- `app/routes/` - Route modules for the active React Router app.
- `app/components/` - Feature and reusable React components.
- `app/components/ui/` - Primitive UI building blocks.
- `app/hooks/` - Shared hooks.
- `app/lib/` - Firebase and shared utilities.
- `app/data/` - Static data modules.
- `app/styles/` - Global styles, tokens, reset, and theme files.
- `raiz/` - Backend workspace for Functions and RAIZ resources.
- `prompts/` - Instructions used by AI-assisted workflows.
