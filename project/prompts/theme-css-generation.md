# theme.css File Generation Rules

- Start with `app/styles/theme.css` when shaping a new visual system.
- Keep the required semantic groups in place: typography, neutral, accent, error, and success.
- Do not remove or rename required variables.
- Prefer semantic variables and existing tokens over ad hoc values.
- Keep the palette consistent with LifePlants: organic, calm, readable, and intentional.
- Verify contrast on light and dark surfaces before finishing.
- Add new semantic variables only when the current set cannot express the design clearly.
