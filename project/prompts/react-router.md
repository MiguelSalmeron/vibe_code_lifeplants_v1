# React Router

- Keep route definitions in `app/routes.ts`.
- Map each URL to a dedicated route module in `app/routes/`.
- Use `index()` for default children and the root page.
- Use `route()` for explicit paths and `layout()` only when a shared shell is real.
- Keep file names in kebab-case and align them with route paths.
- Update documentation when routes or route behavior change.
